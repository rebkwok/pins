Some terminology

Back to top

The PINS website consists of pages, which have a hierarchy. The website home page is the root page, and every other page is a descendent of this page. Think of it like a family tree. In the admin, we build up web pages that will be visible on the live website. If you are an Auction admin, you will only see a limited subset of the website's pages. You'll just see the "Auctions" page, which is a page that lists all the visible auctions. This is the "parent" page for all individual Auction pages. Each Auction is a "child" of this page. Each item in an Auction has its own page (an "Auction Item page"). To create a new auction, you will first create an Auction page. This has some basic auction details - a name, an option description, a start and end date. Then you'll create an Auction Item page for each item in the auction.

Navigating to auctions

Back to top

After logging in to the PINS admin, you'll see a page like this. In the side menu, Pages takes you to the website pages. This is where you create a new auction and add items to it. Auctions contains results of auctions and details of bids.

Click on Pages and the Pages menu opens

Click on the PINS Auctions link; this takes you to the main list of Auctions

Create a new auction

Back to top

Click on the + to add a new auction page. You can also click on the ... to edit the Auctions list page.

This takes you to a form where you can fill out details for the new auction

Add a name for the auction, page content ("body") and the start and end date and time for the auction. Auctions will only be open for bidding between these dates (you can change them later if you need to).

Adding page content

The "body" field allows you to add optional details about the auction. You can add headings and formatting in this field. When you click in the field, a green + will appear to the left of the field. Click on this to display formatting options, such as headings and bullet lists.

If you select text, other formatting options will appear (e.g. bold, italics).

Previewing the page

Click on the screen icon on the right hand side of the top menu to open a panel that shows a preview of how the page will look on the website. Click again to hide it.

Saving as draft vs publishing

At the bottom of the page is a button to save the page. This defaults to "Save draft". If you click on the arrow icon, it will expand to give you the option to Publish instead. A draft page can be previewed by admins, but is not visible on the website yet. A published page is live and visible on the website. You can come back to a page and make edits, and save as draft. This will make the page both Live and draft. The content on the website will not update until you publish the new changes.

If you save a page as draft, you will stay on that page and can continue editing it. If you publish, it will save and take you to the list of Auction pages.

From an Auction page, you can go to the list of Auction Items by clicking on the page name in the top menu. The ... button shows other options, including "Add child page" (i.e. an Auction Item)

Adding auction items

Back to top

From your auction list page (get to it by going to Pages in the side menu, and clicking on your auction's name) add a child page by clicking on the + button.

This opens a form for a new auction item

Add a title and choose a category for the item. If you need a different category, you can add a new one by following these instructions. Fill in other details as provided by the item donor. Make sure the donor email address is correct!

Note: postage needs to be specified by the donor when they submit the item to the auction, so that we can calculate the total amount that the winner will need to pay. If the donor is covering postage, enter 0 in this field.

Adding photos

You can add any number of photos to an auction item (or none). Click on the "add photos" link.

A "choose image" window will pop up.

Click on the "Upload" tab, or on the "upload one now" link. You can now select an image from your computer. Note that you don't need to provide a title - it will use the name of the photo if you leave the title field empty.

Thumbnails of the photos you have uploaded will be displayed on the page. You can add more, remove photos or change the order from here.

Preview auction item page

You can preview an auction item page in the same way as before.

Save page

Save the page as draft or publish it, in the same way as we did for the Auction page.

Viewing live pages

In several places in the admin, you'll see a "Live" link or button. E.g. on the auctions list page below, there are links to view the overall auctions page and the Test Auction page. Clicking on these links will take you to the live page on the website (it will open in a new tab). Note that from this page you can also see the status of each Auction page (draft/live).

Adding a new auction category

Back to top

Click on "Auctions" in the side menu; in the sub-menu, there is an option for Auction Categories.

Click on this and you'll see a list of the available auction categories. Click on the "add auction category" button to add a new open. Warning! You can edit existing auction categories. However, if you do, remember that these will change for any previous auction item that uses them. Only edit categories if you're changing a name for the same things. I.e. it's fine to change "Dog stuff" to "Items for dogs", but you probably don't want to change it to "Items for humans", because any items previously categorised as dog stuff will now be labelled "Items for humans".

Auction results
